Common Issues

Unable to Subscribe

“Account Not In This Store” error message with “Change Store” button

This message may appear when you are currently in a different country than the country of your App Store account. You can try the following step:\

  1. Logout from App Store and iTunes.
  2. Login App Store and Itunes using another account that in the same region of your Apple ID.
  3. Restart your Mac.
  4. Logout from App Store and iTunes. Login again using your own Apple ID.
  5. Try subscribe again.
  6. Try reinstall iCopy if not working.

There is no subscribe info after the purchase is done, restore purchase didn’t work

That’s probably because of network instability when making the purchase. Try delete iCopy, reinstall from App Store, then restore purchase again.

“SKErrorDomain error 0” message

Please try to subscribe again after:

  1. Re-login the Mac App Store.
  2. Re-login the iTunes.
  3. Try reinstall iCopy if not working.

If it doesn’t work, make sure you are running the latest version of macOS, restart your computer, then try again.

paste or copy Issues

If something weird happened when you are copying, try close all the application running except iCopy. If problem solved, there must come conflict with a specific application, find it.

A common situation is the shortcut conflict.
